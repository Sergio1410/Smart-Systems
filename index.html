Páginas de error

Para este apartado, se encuentran las páginas de error que consideramos más importantes en nuestro sitio web.

Error 400

Indica que el servidor no puede procesar la petición debido a un fallo o error de cliente.

Error 403

Indica que no se cuenta con la aprobación necesaria para acceder a una sección web específica.

Error 404

Indica que el host ha sido capaz de comunicarse con el servidor, pero el recurso que se ha pedido no existe.

Error 500

Indica que ha sucedido un error cuando se intentó acceder al servidor.
